# raya-ai-tools-docs

## ai-image-inspector（AI图片解释器）
[Link（链接）](https://github.com/rayafriandion/ai-image-inspector)
> AI Image Inspector can parse raw AI generated images and supports images generated by Comfy, SD, Midjourney, and other methods.

可以将未经压缩或修改过的 AI 生成图片拖入来查看使用了什么 prompt 以及使用了什么 comfyUI Workflow，支持 ComfyUI, Stable Diffusion, Midjourney 等生成出来的图片。

## api-model-list（AI服务API检测）
[Link（链接）](https://github.com/rayafriandion/api-model-list)
> Query, test, and audit any OpenAI-compatible API. Detect bait-and-switch model fraud.

检测 AI 模型服务商具体提供了什么模型，查看该服务商下的模型是否有文字对话、图片识别、工具使用、json输出功能，以及检测是否声明模型和实际提供模型是否一致
